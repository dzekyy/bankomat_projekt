# Bankomat Python Projekt

Simulacija bankomata u Pythonu. Projekat sadrži:
- Modul `bankomat_modul.py` koji implementira klase `Racun`, `Kartica`, `Banka` i funkciju za PIN provjeru.
- Glavni program `glavni.py` koji omogućava korisniku kreiranje računa, kartice i izvršavanje transakcija preko bankomata.

## Zahtjevi
- Python 3.8 ili noviji

## Kako pokrenuti
1. Preuzmite repo.
2. Otvorite terminal u folderu projekta.
3. Pokrenite glavni program:
   ```bash
   python glavni.py
